Action Mailer uses mail gem. Action Mailer is in essence a wrapper around Action Controller and the Mail gem. It provides a way to make emails using templates in the same way that Action Controller renders views using templates. Action Mailer class can also be used to process incoming emails such as a support email which accept tickets using emails. We will talk about this later in this blog post.
