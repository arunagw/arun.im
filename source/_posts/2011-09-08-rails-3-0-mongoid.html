--- layout: post title: Rails 3.0 and Mongoid tags: [] status: draft type: post published: false meta: _edit_last: '1' _su_rich_snippet_type: none --- Hi Guys, Let's talk about creating an Application with Rails and Mongoid. Mongoid is an Object-Document-Mapper for MongoDB which is a NoSQL database. Showing here some steps to get started with a Rails Application with Mongoid. Before that setup MongoDB in your system. You can follow instructions here... MongoDB Start Generate a Rails Application using rails new command and remember to skip activerecord framework. We are going to use Mongoid here. 
rails new mongoid_app --skip-active-record
Change the Gemfile to look like this. Just need to gem Mongoid.! 
gem 'mongoid'
gem 'bson_ext' # Add bson_ext for bson performance.
All done. Generate Mongoid configuration file to connect with MongoDB. 
bundle exec rails generate mongoid:install
Rails Application is ready to boot.. 
bundle exec rails server
Let's create some scaffold for our Application. 
bundle exec rails generate scaffold Author name:string
Author Model will look like 
class Author
  include Mongoid::Document
  field :name, :type => String
end

bundle exec rails generate scaffold Blog title:string body:text
Blog Model will look like 
class Blog
  include Mongoid::Document
  field :title, :type => String
  field :body, :type => String
end
